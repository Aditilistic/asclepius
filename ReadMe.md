## Installation

1) Clone the repository:

```bash
cd /path/where/you/want/to/clone
git clone https://github.com/ChiragAgg5k/asclepius.git
cd asclepius
```

2) Install the required dependencies:

```bash
pip install -r requirements.txt
```

3) Run the main script:

```bash
cd src
python -u main.py
```
